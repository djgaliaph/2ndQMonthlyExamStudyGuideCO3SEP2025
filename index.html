Computer Networking Study Notes

The Definition of a Computer Network

A computer network is a system of interconnected computers and other devices that can share resources and information. These resources can include hardware (like printers) and software (like shared applications or files).

Understanding "LAN"

LAN stands for Local Area Network. It is a network that connects devices within a small, limited geographical area, like a home, school, or office building. A LAN allows connected devices to share resources, such as files, printers, and internet access. [Image of a local area network]

Different Network Types

Networks are categorized by their geographical scope:

  • LAN (Local Area Network): Covers a small area like a building or campus.
  • MAN (Metropolitan Area Network): Covers a city or large campus.
  • WAN (Wide Area Network): Spans a large geographical area, such as a country or the world (e.g., the internet).
  • PAN (Personal Area Network): Covers a very small area, connecting a person's personal devices like a smartphone and a headset.

The Role of a Router

A router is a networking device that acts as a traffic director for data packets. Its primary function is to forward data between different computer networks, such as connecting your home network to the internet. Routers analyze the destination address of a data packet and use a routing table to determine the most efficient path for the data to reach its destination.

Wireless Access Point (WAP)

A Wireless Access Point (WAP) is a device that provides wireless connectivity to a network. It connects to a wired network and creates a wireless local area network (WLAN), allowing devices like laptops, smartphones, and tablets to connect to the network using Wi-Fi signals.

The "NIC"

NIC stands for Network Interface Card. This is a hardware component that allows a computer to physically connect to a network. It provides the circuitry and port (usually an RJ45 port) required to transmit and receive data over a network cable.

Routers and Large Networks

Routers are essential for large networks because they are responsible for directing data traffic between different networks. By using logical addresses (IP addresses), a router determines the most efficient path for a data packet to travel, ensuring it reaches its correct destination, even if it has to cross multiple networks.

UTP Cable Construction

A standard UTP (Unshielded Twisted Pair) cable typically contains four pairs of twisted wires, for a total of eight individual wires. These pairs are color-coded to ensure correct wiring.

The RJ45 Connector

The RJ45 connector is the standard connector used to terminate UTP cables and plug them into network devices. It securely attaches the cable's wires to the pins inside the connector, allowing for a proper connection to a network port.

Coaxial Cable's Shield

A coaxial cable has a central conductor surrounded by an insulating layer, a metallic shield, and an outer jacket. The purpose of the metallic shield is to protect the inner data-carrying wire from external electromagnetic interference (EMI) and radio frequency interference (RFI). This ensures a clean and reliable data signal.

STP vs. UTP

The main difference between Shielded Twisted Pair (STP) and Unshielded Twisted Pair (UTP) cables is the presence of a protective metallic shield in STP. This shield provides additional protection against electromagnetic interference, making STP a better choice in environments with a lot of electrical noise.

Fiber Optic Transmission

Fiber optic cables transmit data using pulses of light. This process is called optical transmission. The light pulses travel through a thin strand of glass or plastic fiber, allowing for very high-speed data transfer over long distances without signal degradation.

The OSI Model

The OSI (Open Systems Interconnection) model is a conceptual framework used to understand and standardize the functions of a telecommunication or computing system without regard to their underlying internal technology and specific protocols. The model divides the communication system into seven abstract layers, from the physical layer (Layer 1) to the application layer (Layer 7).

  • Layer 7: Application Layer (e.g., HTTP, FTP)
  • Layer 6: Presentation Layer (Data formatting, encryption)
  • Layer 5: Session Layer (Managing sessions between applications)
  • Layer 4: Transport Layer (e.g., TCP, UDP)
  • Layer 3: Network Layer (e.g., IP addressing, routing)
  • Layer 2: Data Link Layer (MAC addresses, frames)
  • Layer 1: Physical Layer (Physical media, bits)

IP Address

An IP (Internet Protocol) address is a unique numerical label assigned to each device connected to a computer network that uses the Internet Protocol for communication. An IP address serves two main functions: host or network interface identification and location addressing. There are two main versions: IPv4 (e.g., 192.168.1.1) and IPv6 (a newer, longer format).

Network Topologies

A network topology refers to the physical or logical arrangement of the elements (nodes, links, etc.) of a communication network. Common types include:

  • Bus Topology: All devices are connected to a single central cable (the bus).
  • Star Topology: All devices are connected to a central hub or switch. This is the most common topology in modern networks.
  • Ring Topology: Each device is connected to exactly two other devices, forming a single continuous pathway for signals.

What is a "MAN"?

A Metropolitan Area Network (MAN) is a network that spans a larger area than a LAN but is smaller than a WAN (Wide Area Network). It is typically used to connect multiple LANs within a city or a large campus, providing high-speed connectivity across different buildings or locations.

Ethernet Cabling

The most common type of cable used for a wired Ethernet network is the twisted pair cable. This cable is made of insulated copper wires twisted together in pairs. The twisting helps to reduce electromagnetic interference (EMI) and crosstalk, ensuring a reliable data signal. Two common types are Unshielded Twisted Pair (UTP) and Shielded Twisted Pair (STP).

Network Cable Wiring

The standard wiring color arrangement for a common network cable (T568B) is:

  1. White/Orange
  2. Orange
  3. White/Green
  4. Blue
  5. White/Blue
  6. Green
  7. White/Brown
  8. Brown
